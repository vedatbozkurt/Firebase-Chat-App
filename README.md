# Firebase Chat App
 Realtime Firebase Chat Application with Vue.js

## Installation
[node.js](http://nodejs.org/download/) is required to get ``npm``.


## Firebase setup
Create account on firebase and edit config in src/fire.js file.

```
var config = {
  apiKey: "",
    authDomain: "",
    databaseURL: "",
    projectId: "",
    storageBucket: "",
    messagingSenderId: "",
    appId: ""
};
```

## Project setup
```
npm install
```
